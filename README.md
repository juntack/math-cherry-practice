# さくらんぼ計算 練習アプリ

小学1年生向けに、繰り上がりのある足し算と繰り下がりのある引き算を、さくらんぼ計算で視覚的に学ぶためのアプリです。

## 対象

- 小学1年生
- 繰り上がりのある1桁どうしの足し算を学び始めた児童
- 繰り下がりのある、十何から1桁を引く引き算を学ぶ児童
- タブレット優先、PCブラウザにも対応
- 児童が一人でも操作でき、保護者が横で補助する利用も想定

## 基本方針

このアプリは単なる計算ドリルではなく、数を分けて10のまとまりを作る考え方を学ぶ教材です。

1. 数は複数の数に分けられる
2. 10を作るには、あといくつ必要かを考える
3. 分けた数を用いて、10のまとまりを作る
4. 10と残りを合わせて答えを求める
5. 最終的には、さくらんぼ図なしで暗算できるようにする

## 技術方針

- Vite + React + TypeScript
- Vitest
- タブレット優先のレスポンシブUI
- 初期版は静的Webアプリ
- 将来的にFire HD向けオフラインAPK化を想定

## 現在できること

- 足し算、引き算、どちらも、から出題モードを選べる
- 3もん、5もん、10もん、から問題数を選べる
- やりかたを「さくらんぼ」「あんざん」から選べる
- ヒントのりょうを「おおい」「ふつう」「すくない」から選べる
- 暗算モードではヒントのりょうを無効表示にする
- テンキー型の数字ボタンだけで回答できる
- 正解時は○印と「ピンポーン」音、不正解時は×印と「ブブーッ」音を出す
- 不正解時は答えを表示せず、ヒントを出す
- 現在の問いに合わせて、10フレーム、さくらんぼ図、途中式の表示を切り替える
- 暗算モードでは式だけを表示する
- 練習途中でトップへ戻れる
- 練習中とセッション終了時に正解数を表示する

学習段階1「かずをわける」と段階2「10をつくる」のロジックは残していますが、現在の画面では段階選択を表示せず、段階3「途中計算」を固定で使っています。

## 開発コマンド

```sh
npm install
npm run dev
npm test
npm run build
```

## 初期版で含めないもの

- 成績データ保存
- ログイン機能
- 外部サーバー連携
- 音声読み上げ
- ドラッグ操作
- 10フレームの本格表示と詳細なアニメーション
