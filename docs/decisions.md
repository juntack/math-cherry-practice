# 決定事項・未確定事項

## 決定事項

- 技術構成は Vite + React + TypeScript + Vitest
- タブレット優先で設計する
- PCブラウザにも対応する
- 初期版の入力は数字ボタンのみ
- 将来的にドラッグ操作を検討する
- 問題数は可変設定にする
- MVPでは `3問 / 5問 / 10問` の選択式を基本にする
- 引き算の初期方式は「十何を10と一の位に分ける方式」
- 10フレームはMVPで最小表示を実装し、本格表示やアニメーションは将来対応にする
- ヒントは段階的に具体化するが、答えは表示しない
- 正解時は「ピンポーン」音と○印
- 不正解時は「ブブーッ」音と×印
- 出題モードは「たしざん」「ひきざん」「どちらも」をラジオボタンで選択する
- 問題数は「3もん」「5もん」「10もん」を選択する
- ヒントのりょうは「おおい」「すくない」を選択する
- 段階1、段階2のロジックは残すが、現在のUIでは段階3を固定で使う
- GitHubリポジトリは `git@github.com:juntack/math-cherry-practice.git`
