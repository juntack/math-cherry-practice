# MVP範囲

## MVPの目的

児童が、さくらんぼ計算の基本的な流れに沿って、足し算・引き算の問題を解けることを確認する。

## 含めるもの

- 足し算モード
- 引き算モード
- 混合モード
- 問題数設定
- 段階1: 数の分解
- 段階2: 10を作る分解
- 段階3: 途中計算
- さくらんぼ図表示
- 数字ボタンによる回答
- 正誤判定
- 誤答分類
- ヒント表示
- 正解時の○印と効果音
- 不正解時の×印と効果音
- セッション終了画面
- 問題生成ロジックの単体テスト
- 誤答分類とヒント生成の単体テスト

## 含めないもの

- 成績保存
- ユーザー別管理
- 音声読み上げ
- ドラッグ操作
- 10フレームの本格表示
- 暗算モード
- 保護者向け詳細レポート
- 外部サーバー連携
- GitHubリポジトリ作成

## MVP完了条件

- 足し算、引き算、混合を選択できる
- 問題数を選択できる
- 児童が数字ボタンのみで回答できる
- 不正解時に答えを表示せず、ヒントを出せる
- 問題生成、ステップ生成、正誤判定、ヒント生成がUIから独立している
- 主要ロジックに単体テストがある
