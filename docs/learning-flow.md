# 学習フロー

## 全体フロー

1. ホーム画面を開く
2. 出題モードを選ぶ
3. 問題数を選ぶ
4. ヒントのりょうを選ぶ
5. 問題が表示される
6. 数字ボタンで回答する
7. 正解なら次のステップへ進む
8. 不正解ならヒントを表示し、再回答する
9. 問題の全ステップが終わったら次の問題へ進む
10. 指定問題数が終わったら終了画面を表示する

現在の画面では学習段階を選ばず、段階3「途中計算」を固定で使う。

## 段階1: 数の分解

目的は、1つの数を2つに分ける感覚を身につけること。

例:

```txt
7は、2といくつ？
```

さくらんぼ図:

```txt
  7
 / \
2   □
```

## 段階2: 10を作る分解

目的は、「あといくつで10か」を考え、加える数または分ける数を決めること。

足し算例: `9 + 6`

- `9` を10にするにはあと `1`
- `6` を `1` と `5` に分ける
- この段階では最終答えは問わない

## 段階3: 途中計算

目的は、分解から答えまでの流れを順番に理解すること。

足し算例: `8 + 7`

1. `8` を10にするにはあといくつか問う
2. `7` を `2` といくつに分けるか問う
3. `8 + 2 = 10` を確認する
4. `10 + 5 = 15` を問う

引き算例: `13 - 8`

1. `13` を `10` といくつに分けるか問う
2. `10 - 8 = 2` を問う
3. `2 + 3 = 5` を問う
